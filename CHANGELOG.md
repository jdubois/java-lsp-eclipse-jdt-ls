# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [1.0.0] - 2026-03-13

### Added

- Initial release
- LSP configuration for Eclipse JDT Language Server
- Auto-download of latest JDTLS milestone on first use
- Cross-platform launcher: bash script (macOS/Linux), PowerShell script (Windows)
- Java 21+ detection and version validation
- Platform auto-detection (`config_mac`, `config_linux`, `config_win`)
- JDTLS setup skill for installation guidance and troubleshooting
- Support for `JDTLS_HOME`, `JAVA_HOME`, and `JDTLS_DATA_DIR` environment variables
